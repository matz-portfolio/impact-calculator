# Impact Calculator — One for the World

Interactive impact calculator developed as a B-level thesis and volunteer project for One for the World.

The tool helps users explore estimated charitable impact through donation allocation logic and a clear, user-facing React interface. The project was later published on One for the World’s website.

## Project Context

This repository contains my portfolio fork/version of the Impact Calculator project developed for One for the World.

The project was created as part of a university B-level project and volunteer collaboration with One for the World. The aim was to build a practical web tool that presents impact estimates in an accessible and interactive way.

## Tech Stack

- React
- JavaScript
- Vite
- HTML
- CSS

## Features

- Responsive React interface
- Donation allocation sliders
- Live impact summary
- Currency and age/lifetime support
- Optional email gate before showing results
- Static-site deployment support

## Repository Structure

- `impact-calculator-react/` – current React/Vite version of the project
- `impact_calculator/` – legacy version kept for reference
